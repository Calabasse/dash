Prescription Tuning

Edit the parameters the workout engine (prescribe_hypertrophy.py) uses. Pick a philosophy, tweak any value, Save to a config file. Deterministic algorithm — no AI.
This page is static — it can't run the Python live. Save → replace prescription_config.json in the repo → redeploy (or drop it where deploy.sh reads it). Prescriptions update on the next build.

Training philosophy

Global parameters

warm-up fracs = comma list of % of top load (one per warm-up set). backoff frac = backoff load as a fraction of top. step sizes = how much load is added when a set is "earned".

Per-movement parameters

movement typerep lorep hi anchortop RPE warmupsbackoffs isolation
anchor = reps you must hit on the top set before load goes up. top RPE = the intensity cap. Loads always snap to your real equipment (plates / bands) regardless of these.

Save & load

Save downloads the full config with your edits and the selected philosophy as active. To apply: replace prescription_config.json in the dash-site repo and redeploy.
